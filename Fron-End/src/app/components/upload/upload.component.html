Notes and steps that must be followed to take note of the lifting process and everything related to it:

- First select the field you want to talk about and upload contents in it.

- The domain being identified has a clear learning objective.

- Each "domain" contains a set of courses.

- Each course contains full educational content (full play list).

- Each course has a set of lessons.

- Each lesson contains a specific content that aims to communicate a specific goal related to the course material.

- Each lesson contains a set of resources from "pdf, power point...", which help in the achievement and comprehension of the scientific material and the arrangement of the information well.
